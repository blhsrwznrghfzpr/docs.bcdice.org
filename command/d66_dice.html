D66

D66

D66
D66A
D66D
D66N

6面ダイスを2個ダイスロールして、それぞれを十の位と一の位とした値を表示します。

出目の大小によって位を入れ替えることができます。出目の入れ替えは以下の3種類です。

  • 昇順：大きい出目を一の位にする
  • 降順：大きい出目を十の位にする
  • そのまま：出目の大小による位の入れ替えを行わない

コマンドと入れ替えの対応表

コマンド 入れ替えの種類 備考
D66A 昇順 Ascending order の A
D66S 昇順 Sort の S
D66D 降順 Descending order の D
D66N そのまま No sort の N
D66 - ゲームシステムの設定によって変化。
基本的にはそのままに設定されている。

実行例

[DiceBot]> D66A
(D66A) ＞ 36

[DiceBot]> D66D
(D66D) ＞ 41

[DiceBot]> D66N
(D66N) ＞ 25

[DiceBot]> D66N
(D66N) ＞ 45
